CutImage
JavaScript библиотека для обрезки изображений
Удобство
API модуля очень удобно для использования: много настраеваемых параметров, модуль базируется на ваших DOM элементах контроль над которыми остается за вами.
Нативность
Модуль использует только JavaScript и вам не нужно подключать никаких сторонних библиотек.
Быстродействие
Благодаря использованию 'чистого' JavaScript модуль не загружает CPU пользователей сайта и все довольны.
Примеры
Пример 1, прямоугольная область выделения 6:3.
Код примера
var img1Cut = new CutImage(img1, { size: '6:3', preview: [{ el: preview1, isEllipse: false, width: 200 }] });
Пример 2, овальная область выделения 4:3.
Код примера
var img2Cut = new CutImage(img2, { size: '4:3', preview: [{ el: preview2, isEllipse: true, width: 300 }] });
